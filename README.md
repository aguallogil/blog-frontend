# BlogFrontend

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) version 17.0.5.

## Install dependencies
Ejecuta `npm install` para instalar la carpeta node_modules.

## Development server

Ejecuta `npm run start:dev` para ejecutar la aplicacion con el --watch activado verificando cambios. La aplicacion se ejecutar√° en  `http://localhost:4200/`, iniciar sesion con username 'admin' y password '1234'.

## Ejecutar tests

Ejecuta `npm run test` para ejecutar los test e2e en cypress.
